## Installation

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/a-mythh/MoM-Generator-Frontend.git
    cd MoM-Generator-Frontend
    ```

2. **Install Dependencies**:
    ```bash
    npm install
    ```

3. **Run the App**:
    ```bash
    npm run dev
    ```

## Usage

1. Open the website.
2. Upload the audio file.
3. Get the JSON file of the transcription.
